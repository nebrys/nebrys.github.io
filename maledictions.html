MALÉDICTIONS

  1. Estomac sans fond : Triple vos besoins en nourriture.
  2. Phobie : 1d4 : peur d'être seul, peur d'être dans un espace clos, peur du noir, peur des hauteurs.
  3. Détesté par les chiens : Les chiens vous attaquent à vue.
  4. Mites boursières : Perdez 1d100% de votre argent régulièrement.
  5. Fragile : 1d4 PV au lieu de 1d6.
  6. Dents hurlantes : Vos dents crient lorsque vous parlez.
  7. Maudit par le feu : Double les dégâts subis par le feu.
  8. Trahi : Les dégâts infligés par un allié sont triplés.
  9. Hallucinations : Vos sens vous trahissent. Le MJ vous donne de mauvaises descriptions quand vous êtes seul, toujours surpris en combat si seul.
  10. Cauchemars : 3 chances sur 6 de bien dormir la nuit, 4 chances sur 6 si alcoolisé.
  11. Accablé : Vous êtes abusif, parano, irrationel ou autre. Ceux qui passent une journée avec vous font un jet de santé mentale et perdent 1 point si échec.
  12. Âme magnétique : Les sorts ont tendances à se diriger vers vous.
  13. Lisible : Vos pensées de surfaces sont écrites sur votre corps.
  14. Nervosité permanente : Doublez les pertes de santé mentale
  15. Cavalier de l'apocalypse : Contractez 3 maladies.
  16. Régime inquiétant : Peut uniquement se nourrir de viande crue, violemment malade autrement.
  17. L'Élu : Si quelquechose doit arriver à un membre aléatoire du groupe, ça arrive à vous.
  18. Sans espoir : Désavantage sur tous les jets jusqu'à ce que vous en réussissiez un. Tous les jours.
  19. Double maléfique : Il apparaît très loin, il sait où vous vous trouvez.
  20. Lesté : Vous coulez instantanément quand dans du liquide.
